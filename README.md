## ðŸš€ Getting Started

### Prerequisites
- A modern web browser.
- A **Google Gemini API Key**. You can obtain one at [Google AI Studio](https://aistudio.google.com/).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/caliplayer.git
   cd caliplayer
   ```

2. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add your API key:
   ```env
   API_KEY=your_gemini_api_key_here
   ```

3. **Install dependencies and start:**
   This project uses ESM modules directly. If you are using a standard dev server (like Vite or Live Server):
   ```bash
   npm install
   npm run dev
   ```
